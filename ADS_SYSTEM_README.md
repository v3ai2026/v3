# AI 全自动广告投放和营销系统

## 系统概览

这是一个**完全自动化的 AI 广告营销系统**，整合了多个主流广告平台，实现从广告创意生成、投放、优化到转化的全流程自动化。

## ✅ 已实现功能

### 1. 多平台广告投放系统

支持的广告平台：
- ✅ Google Ads（搜索/展示/购物/YouTube）
- ✅ Facebook Ads（Facebook + Instagram）
- ✅ TikTok Ads（抖音国际版）
- ✅ 抖音广告（国内版，含直播数据）
- ✅ 快手广告（短视频营销）
- ✅ 小红书广告（KOL 合作、种草营销）
- ✅ 微信广告（朋友圈/公众号/小程序）
- ✅ 百度推广（搜索广告、信息流）
- ✅ 腾讯广告（QQ、QQ空间、腾讯视频）
- ✅ 阿里妈妈（淘宝直通车、超级推荐）

广告类型：
- ✅ 搜索广告（关键词竞价）
- ✅ 展示广告（横幅/原生广告）
- ✅ 视频广告（YouTube/抖音/快手）
- ✅ 购物广告（Google Shopping）
- ✅ 动态广告（DPA - 动态产品广告）
- ✅ 再营销广告（Retargeting）
- ✅ 应用推广广告

## 🚀 使用方法

### 1. 启动应用
```bash
npm install
npm run dev
```

### 2. 访问广告系统
点击侧边栏的 "📢 Ads" 图标进入广告管理系统

### 3. 创建广告
1. 点击 "创建广告" 按钮
2. 输入产品信息
3. 选择投放平台
4. 设置预算
5. AI 自动生成创意
6. 选择最佳版本
7. 一键启动广告

### 4. 监控和优化
- 仪表板实时显示所有广告表现
- AI 自动提供优化建议
- 点击 "AI 自动优化" 自动优化所有广告

## 📄 API 文档

### UnifiedAdsService
主要的广告管理服务

```typescript
// 创建广告活动
await adsService.createCampaign({
  name: "夏季促销",
  platform: "google_ads",
  budget: { daily: 500, total: 15000 }
  // ...更多配置
});

// 获取聚合指标
const metrics = await adsService.getAggregateMetrics();

// AI 自动优化
const results = await adsService.autoOptimizeCampaigns();
```

### AICopywritingService
AI 文案生成服务

```typescript
// 生成广告文案
const copy = await copywritingService.generateAdCopy({
  productName: "连衣裙",
  productDescription: "舒适透气的夏季连衣裙",
  targetAudience: "25-45岁女性",
  sellingPoints: ["透气", "时尚", "限时7折"],
  platform: "google_ads"
});
```

### MarketingAutomationService
营销自动化服务

```typescript
// 创建自动化工作流
const workflow = await automationService.createWelcomeFlow();

// 触发工作流
await automationService.triggerWorkflow(workflow.id, userId);
```
